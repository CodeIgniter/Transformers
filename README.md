Transformers
============

基于 Javascript 的组件化开发框架，主要关注组件路由、组件消息传递和组件异步加载等，其中要解决的核心问题是组件间的解耦。

如何安装
------------

```bash
bower install transformers
```

如何构建
------------

从 git 上克隆一个副本：

```bash
git clone git://github.com/hex-ci/Transformers.git
```

进入 Transformers 目录，然后执行构建脚本：

```bash
cd Transformers && npm run build
```

构建的文件将放在 `dist/` 目录下。
